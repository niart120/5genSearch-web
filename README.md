Gen5 RNG web app (React + Rust/WASM)

## データの出典

### エンカウントデータ

本アプリケーションのエンカウントテーブルデータは、[ポケモンの友](https://pokebook.jp/)（pokebook.jp）のデータを元に加工・再構成したものです。

- 出典: [ポケモンの友 - データ集](https://pokebook.jp/data/)
- 引用・転載条件: [情報の引用・転載・転用などされる場合のお願い](https://pokebook.jp/data/reffer)

### 参照実装

本プロジェクトのアーキテクチャは [niart120/pokemon-gen5-initseed](https://github.com/niart120/pokemon-gen5-initseed) を参照し、別設計として再構成したものです。
